# ai-study-focus-monitor
AI-based real-time focus detection and cognitive refresh system using OpenCV.
Overview

This project explores real-time attention monitoring using computer vision. The system detects prolonged eye closure and triggers cognitive refresh tasks to restore focus.

Technologies Used

Python

OpenCV

Tkinter

Features

Real-time face and eye detection

Distraction threshold logic (3-second rule)

Cognitive refresh puzzle system

Cooldown mechanism to prevent repeated triggering

Future Improvements

Focus percentage tracking

Session analytics

Emotion detection integration
